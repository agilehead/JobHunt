About Jobhunt.in

What is jobhunt.in?

Jobhunt.in is the easiest job site in India, helping people find jobs by simply uploading resumes. These resumes are emailed to companies that have matching job openings. Learn More.

Some users prefer to hunt jobs privately, and to expose resumes only to select recruiters. They can opt for an Anonymous Account, which costs as little as Rs.200 a year.
Learn more about Anonymous Accounts.

How does it work for Recruiters?

Recruiters can create Resume Subscriptions by mentioning a search criteria, and receive a daily report on fresh resumes matching that criteria.
See how it works.
